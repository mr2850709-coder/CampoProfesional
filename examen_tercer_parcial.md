# Comandos Basicos de terminal 
pwd: Muestra la ruta del directorio actual.

ls: Lista archivos y carpetas del directorio.

ls -l: Lista detallada (permisos, fechas, tamaño).


cd nombre_carpeta: Entra a una carpeta.

cd ..: Regresa un nivel.

clear: Limpia la terminal.

history: Muestra el historial de comandos.

# Comandos Basicos de Git/GitHub
git status: 
Muestra archivos modificados y pendientes.

git add archivo:	Añade un archivo al área de staging.

git add .  : Añade todos los archivos modificados.

git commit -m "mensaje":	Guarda los cambios en la historia.

git push:
Sube cambios futuros.

ls: Ver contenido de los directorios

ls -la: Contenido y permisos de los directorios

mkdir: Crear una carpeta

touch: Crear un archivo

vim datos.txt: abrir y editar un archivo en terminal
